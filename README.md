# encrypt_dic
字典加密
#### 场景
在密码爆破中经常会遇到提交的payload被加密，这种一般都是前端JS加密，所以写了一个脚本将字典通过前端加密的方法进行处理。
当前脚本仅用于字典的加密转换。<br />

#### 安装execjs包
 `  pip install PyExecJS`
#### 使用方法
本项目请使用python3执行<br />
` python3 encrypt_dic.py <js_file_path> <js_function_name> <input_file_path> <output_file_name>`
